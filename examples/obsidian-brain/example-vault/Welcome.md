# Welcome — RuVector Brain demo vault

This vault exists to demo the [RuVector Brain](https://github.com/ruvnet/RuVector) Obsidian plugin. Once the plugin is enabled and the local brain/embedder are running, try:

- `Cmd+Shift+B` → semantic search
- Ribbon icon → open the **Related memories** side panel
- Command palette → `Brain: Bulk-sync vault → brain`
- Command palette → `Brain: DPO — mark current note as chosen`

The frontmatter `brain-category:` key above steers how this note is categorised when indexed. Edit it, save, and the plugin re-indexes automatically (if auto-index is enabled).
